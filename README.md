## Passo 2: Iniciando um repositório

```bash
git config --global user.name = "Seu Nome"
git config --global user.email = "seu@email.com"
```

```bash
git config user.name
git config user.email
```

```bash
git init
```

```bash
git add .
```

```bash
git commit -m 'o começo'
```
